******************************例程详细描述*****************************
功能描述:
	本例程描述了FSMC模拟8080时序来实现LCD的触摸功能，主要涉及以下几点内容：
    1. FSMC采用FSMC NOR/PSRAM 中的模式2时序来模拟8080时序，该时序与LCD用到的8080时序及其相似；
    2. 所用到的LCD屏，自身集成了了液晶控制器，控制芯片为ILI9341，该芯片自带显存；
    3. 所用到的LCD屏为电阻屏，通过XPT2046芯片来获取屏幕按压信息，MCU通过SPI与XPT2046芯片进行通信；
    4. 本例用到的是FSMC NOR/PSRAM bank1的第一小块，地址空间为0X6000 0000 ~ 0X63FF FFFF，
       如果选择其它的NE引脚，则对应的地址空间不同。
    5. FSMC_A16 接LCD的DC(寄存器/数据选择)脚，通过FSMC_A16不同值来实现命令与数据的切换。
    6. 在本例中使用的是 16 位的数据访问方式，所以 HADDR 与 FSMC_A 的地址线连接关系会左移一位，FSMC_A16
       为 1 时，实际上内部地址的第 17 位为 1。所以本例中用：向0x60400000地址写入值表示命令，向0x60420000写入值表示数据。
        
时钟配置:
    本例程采用HSE+PLL72MHz作为系统时钟，用户如果需要更改系统时钟配置，请参考对应用户手册或者
	RCC库函数及RCC相关例程。

适用芯片:
	HK32F103VCT6/HK32F103VDT6/HK32F103VET6/HK32F103V8T6A/HK32F103VBT6A

涉及外设:
	FSMC/RCC/GPIO
	
涉及管脚:
    LCD控制信号：    
    LCD_RD(FSMC_NOE) --> PD4
    LCD_WR(FSMC_NWE) --> PD5
    LCD_CS(FSMC_NE1) --> PD7
    LCD_RS(FSMC_A16) --> PD11
    LCD_INT(PENIRQ)  --> PC1-->PE4
    LCD_BL           --> PB11-->PD12
    
    
    XPT2046控制信号：
    LCD_SCLK(DCLK) --> PE0
    LCD_MISO(DOUT) --> PE3
    LCD_MOSI(DIN)  --> PE2
    LCD_SCS(CS#)   --> PD13
    

注意事项:
	1. 推荐使用MDK5.00及其以上版本编译本工程（MDK-ARM）
	          或者IAR8.22及其以上版本编译本工程(EWARM);
	2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
	1. 直接编译程序下载到评估版，重启评估版;
	2. 程序上电会先进行校准，校准完成以后可在屏幕输入内容;
