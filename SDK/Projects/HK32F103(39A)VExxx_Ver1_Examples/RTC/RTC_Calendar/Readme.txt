******************************例程详细描述*****************************
功能描述:
	本例程演示了如何使用RTC外设。
	1. RTC时钟选为LSE，RTC时钟可以通过PC13输出，通过宏 RTCClockOutput_Enable 配置。
	2. RTC处于备份(BKP)域，当VDD关闭时，RTC仍然由VBAT供电，因此如果电池连接到VBAT引脚上，RTC的配置不会丢失。
     在备份数据寄存器1(BKP_DR1)中写入一个标志值，以指示RTC是否已经配置。
	3. 上电后，程序检测BKP_DR1中是否有值，如果没有，代表RTC未初始化，此时初始化RTC并且向BKP_DR1中写入标志值。
	   如果BKP_DR1中存在标志值，则RTC无需初始化。
	4. 在RTC秒中断中，翻转LED2。
	5. RTC每隔1S，通过串口输出当前最新时间（时、分、秒）。
	
时钟配置:
	本例程采用HSE+PLL72MHz作为系统时钟，用户如果需要更改系统时钟配置，请参考对应用户手册或者
	RCC库函数及RCC相关例程。

适用芯片: 
	HK32F103xC,HK32F103xD,HK32F103xE,HK32F103xxxxA全系列

涉及外设:
	BKP/RTC/RCC/USART
	
涉及管脚:	
	VBAT
	PC13 -------> RTC时钟64分频输出
	PB0 --------> LED2
	PA9 --------> USART_TX

注意事项:
	1. 推荐使用MDK5.00及其以上版本编译本工程（MDK-ARM）
	          或者IAR8.22及其以上版本编译本工程(EWARM);
	2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
	1. 直接编译程序下载到评估版，重启评估版;
	2. 上电后，通过串口助手输入当前时间，看LED2是否正常闪烁，PC13输出是否正常，串口是否正确输出当前时间。
	   手动NREST或者Vbat供电，VDD掉电再上电后，确认RTC时钟是否保持。
