******************************例程详细描述*****************************
功能描述:
	本例程演示了如何使用RTC的自动唤醒定时器从standby模式唤醒。
	1. 上电后，MCU处于运行模式，此时LED2快闪；
	2. 当KEY1按键按下后，使能自动唤醒定时器（5s倒计时），进入standby模式，此时LED2熄灭；
	3. 当5s后，自动唤醒定时器使MCU唤醒，此时LED2慢闪； 
	
时钟配置:
	本例程采用HSE+PLL72MHz作为系统时钟，用户如果需要更改系统时钟配置，请参考对应用户手册或者
	RCC库函数及RCC相关例程。

适用芯片: 
	HK32F103xC,HK32F103xD,HK32F103xE,HK32F103xxxxA全系列

涉及外设:
	PWR/RTC/RCC/USART
	
涉及管脚:	
	
	PA0 --------> KEY1
	PB0 --------> LED2
	PA9 --------> USART_TX

注意事项:
	1. 推荐使用MDK5.00及其以上版本编译本工程（MDK-ARM）
	          或者IAR8.22及其以上版本编译本工程(EWARM);
	2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
	1. 直接编译程序下载到评估版，重启评估版;
	2. 上电后，观察LED2是否快闪，按下KEY1按键，观察串口是否输出“Enter standby mode”，
	   并且LED2熄灭，等5s后，串口是否输出“Standby Reset Wakeup”并且LED2慢闪。
