******************************例程详细描述*****************************
功能描述:
    本例程描述了使用3个ADC在独立模式下的AD转换，其中ADC1和ADC3使用DMA对
    采样数据进行搬运，ADC2使用EOC中断事件对数据进行搬运，3个ADC的电压采
    样值通过USART周期性（500ms）的打印。

时钟配置:
    本例程采用HSE72MHz作为系统时钟，用户如果需要更改系统时钟配置，可通过
    SYSCLK_SOURCE宏定义来选择不同的时钟源。

适用芯片:
    HK32F103xC,HK32F103xD,HK32F103xE,HK32F103xxxxA全系列

涉及外设:
    ADC/DMA/USART/GPIO

涉及管脚:
    ADC相关信号：
    ADC_Channel10  -- > PC0
    ADC_Channel13  -- > PC3    
    ADC_Channel12  -- > PC2
    USART相关信号：
    USART1_TX --> PA9
    USART1_RX --> PA10

注意事项:
	1. 推荐使用MDK5.00及其以上版本编译本工程（MDK - ARM）
       或者IAR8.22及其以上版本编译本工程(EWARM);
	2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
	1. 直接编译程序下载到评估版，重启评估版;
	2. 调整PC0/PC3和PC2的输入电压，观察串口打印输出值。
