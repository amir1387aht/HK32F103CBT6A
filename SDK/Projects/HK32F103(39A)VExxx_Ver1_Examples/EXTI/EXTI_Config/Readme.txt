******************************例程详细描述*****************************
功能描述:
	本例程描述了如何配置外部中断线。本例程配置了2条EXTI线(EXTI0(PA0)、EXTI1(PA1))，下降沿产生中断。
	在中断程序中打开或者关闭LED1。
	
时钟配置:
	本例程采用HSE+PLL72MHz作为系统时钟，用户如果需要更改系统时钟配置，请参考对应用户手册或者
	RCC库函数及RCC相关例程。

适用芯片:
	HK32F103xC,HK32F103xD,HK32F103xE,HK32F103xxxxA全系列

涉及外设:
	EXTI/GPIO/NVIC/RCC
	
涉及管脚:	
	PA0 ---------> KEY1
	PA1 ---------> KEY2
	PB1 ---------> LED1 

注意事项:
	1. 推荐使用MDK5.00及其以上版本编译本工程（MDK-ARM）
	          或者IAR8.22及其以上版本编译本工程(EWARM);
	2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
	1. 直接编译程序下载到评估版，重启评估版;
	2. 给PA0或PA1下降沿(按下key1/key2)，观察LED1是否点亮或者熄灭
