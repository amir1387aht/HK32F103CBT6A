******************************例程详细描述*****************************
功能描述:
本例程描述了如何使用DAC外设来生成几个波形。当按下按键时，DAC通道1/2将在
以下两种输出波形之间切换。
	-正弦波形(通道1)及自动扶梯波形(通道2)。
	-噪音波形(通道1)及三角波形(通道2)。
        
时钟配置:
    本例程采用HSE72MHz作为系统时钟，用户如果需要更改系统时钟配置，可通过
	SYSCLK_SOURCE宏定义来选择不同的时钟源。

适用芯片:
	HK32F103xC,HK32F103xD,HK32F103xE,HK32F103xxxxA全系列

涉及外设:
	DAC/DMA/GPIO
	
涉及管脚:
    DAC相关信号：
	DAC_Channel1  --> PA4
	DAC_Channel2  --> PA5
注意事项:
	1. 推荐使用MDK5.00及其以上版本编译本工程（MDK-ARM）
	          或者IAR8.22及其以上版本编译本工程(EWARM);
	2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
	1. 直接编译程序下载到评估版，重启评估版;
	2. 使用示波器观察DAC通道1/2的输出波形，按下按键，DAC输出通道1/2将切换
	   输出波形。
