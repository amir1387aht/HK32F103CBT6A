******************************例程详细描述*****************************
功能描述:
	本例程描述了同步模式配置使用。
	通过调用库函数HK32F103xC,HK32F103xD,HK32F103xE,HK32F103xxxxA_USART来完成对USART的Synchronous配置，BPS为115200，
	1.通过USART1发送数据到SPI1，然后比较数据一致性。比较结果存在TransferStatus1。
	2.再通过SPI1发送数据到USART1,比较数据一致性。比较结果存在TransferStatus2。
	
时钟配置:
	本例程采用HSE8M_PPL72MHZ作为系统时钟，用户如果需要更改系统时钟配置，需要在
	Usart_Config函数之前完成，关于系统时钟如何配置，请参考对应用户手册或者
	RCC库函数及RCC相关例程。

适用芯片:
	HK32F103xC,HK32F103xD,HK32F103xE,HK32F103xxxxA全系列

涉及外设:
	USART1
	SPI1
涉及管脚:	
	PA5--->PA8
	PA6--->PA10
	PA7--->PA9
注意事项:
	1. 推荐使用MDK5.00及其以上版本编译本工程（MDK-ARM）
	          或者IAR8.22及其以上版本编译本工程(EWARM);
	2. 使用之前检测电源VCC、GND等是否正常。
如何使用:
	1. 直接编译程序下载到评估版，重启评估版;
	2. 如果TransferStatus1为PASSED则SPI1接收的数据正确
	3. 如果TransferStatus2为PASSED则USART1接收的数据正确
