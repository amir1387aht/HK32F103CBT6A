******************************例程详细描述*****************************
功能描述:
	通过调用库函数HK32F103xC,HK32F103xD,HK32F103xE,HK32F103xxxxA_USART来完成对USART的MultiProcessor配置，BPS为115200，
	通过USART2发送数据到USART3，后比较数据一致性。一致则LED1，LED2,LED3成流水灯
	状闪烁。
	
时钟配置:
	本例程采用HSE8M_PPL72MHZ作为系统时钟，用户如果需要更改系统时钟配置，需要在
	Usart_Config函数之前完成，关于系统时钟如何配置，请参考对应用户手册或者
	RCC库函数及RCC相关例程。

适用芯片:
	HK32F103xC,HK32F103xD,HK32F103xE,HK32F103xxxxA全系列

涉及外设:
	USART2
	USART3
涉及管脚:	
	PD5---->PC11
	PD6---->PC10

注意事项:
	1. 推荐使用MDK5.00及其以上版本编译本工程（MDK-ARM）
	          或者IAR8.22及其以上版本编译本工程(EWARM);
	2. 使用之前检测电源VCC、GND等是否正常。
如何使用:
	1. 直接编译程序下载到评估版，重启评估版;
	2. 通过LED闪烁判断USART传输数据是否正确;
