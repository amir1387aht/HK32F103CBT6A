******************************例程详细描述*****************************
功能描述:
	本例程描述了如何配置进入和唤醒Standby mode。
	通过调用库函数PWR_EnterSTANDBYMode()来完成对Standby mode的配置，程序运行后约5s进入Standby 
	通过wake up pin PA0产生一个下降沿来唤醒Standby mode，此时系统复位，5s后又重新进入Standby ，每次唤醒后点亮LED1。

	
时钟配置:
	本例程采用HSE+PLL72MHz作为系统时钟，用户如果需要更改系统时钟配置，请参考对应用户手册或者
	RCC库函数及RCC相关例程。

适用芯片:
	HK32F103xC,HK32F103xD,HK32F103xE,HK32F103xxxxA全系列

涉及外设:
	RCC/PWR/GPIO/SYSTICK
	
涉及管脚:	
	PA0 ---------> WKUP0 IO	
	PB1 ---------> LED1

注意事项:
	1. 推荐使用MDK5.00及其以上版本编译本工程（MDK-ARM）
	          或者IAR8.22及其以上版本编译本工程(EWARM);
	2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
	1. 直接编译程序下载到评估版，重启评估版;
	2. 下载程序后约5s进入standby，通过PA0产生一个上升沿唤醒standby，同时点亮LED1
