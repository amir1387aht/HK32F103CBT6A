******************************例程详细描述*****************************
功能描述：
	本例程描述了如何将JTAG IO用做GPIO。
	禁用SWJ-DP。SWJ-DP引脚配置为推挽输出。
	PA13(JTMS/SWDAT)， PA14(JTCK/SWCLK)， PA15(JTDI)，PB3(JTDO)和PB4(JTRST)引脚在无限循环中翻转。
	JTAG IO被禁用，芯片无法与调试器建立连接，需要将NRST引脚电平拉低芯片处于复位状态，才能重新建立连接

时钟配置：
	本例程采用HSE8M+PLL = 72MHz作为系统时钟，用户如果需要更改系统时钟配置，
	关于系统时钟如何配置，请参考对应用户手册或者RCC库函数及RCC相关例程。

适用芯片：
	HK32F103xC,HK32F103xD,HK32F103xE,HK32F103xxxxA全系列

涉及外设：
	GPIO
	RCC
	AFIO

涉及管脚:	
	PA13
	PA14
	PA15
	PB3
	PA4

注意事项：
	1. 推荐使用MDK5.00及其以上版本编译本工程（MDK-ARM）
	          或者IAR8.22及其以上版本编译本工程(EWARM)；
	2. 使用之前检测电源VCC、GND等是否正常。

如何使用：
	1. 直接编译程序下载到评估版，重启评估版；
	2. 通过示波器观察JTAG引脚的波形，是否复用为GPIO；
