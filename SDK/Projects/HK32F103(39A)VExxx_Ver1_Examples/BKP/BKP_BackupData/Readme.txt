******************************例程详细描述*****************************
功能描述:
	本例程演示了如何将用户数据存储在Backup数据寄存器中。
	1. 由于Backup域在关闭VDD的情况下，仍然由VBAT供电，因此在VBAT供电的情况下，即使VDD掉电，Backup域
	的内容也不会丢失。
	2. 程序开始时，检查验证板是否上电复位。如果是，则检查BKP数据寄存器中的值。
	3. 非上电复位，不检测BKP数据寄存器中的值。
	
时钟配置:
	本例程采用HSE+PLL72MHz作为系统时钟，用户如果需要更改系统时钟配置，请参考对应用户手册或者
	RCC库函数及RCC相关例程。

适用芯片:
	HK32F103xC,HK32F103xD,HK32F103xE,HK32F103xxxxA全系列

涉及外设:
	BKP/RCC/USART
	
涉及管脚:	
	VABT
	PA9 --------> USART_TX

注意事项:
	1. 推荐使用MDK5.00及其以上版本编译本工程（MDK-ARM）
	          或者IAR8.22及其以上版本编译本工程(EWARM);
	2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
	1. 直接编译程序下载到评估版，重启评估版;
	2. 在VBAT供电时，检测Backup数据寄存器在VDD掉电后是否依旧保持。
