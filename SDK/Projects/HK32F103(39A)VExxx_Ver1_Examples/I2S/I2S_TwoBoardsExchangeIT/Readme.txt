******************************例程详细描述*****************************
功能描述:
	描述了I2S两板间中断通信，I2S1中断发送到I2S2。I2S2中断接收
	并比较是否正确。
	1.主板用I2S1,按下按键后开始中断发送数据。
	2.从板用I2S2，中断接收数据并比较，正确亮起灯1，错误亮起灯2。

	
时钟配置:
	本例程采用HSE8M_PPL72MHZ作为系统时钟，用户如果需要更改系统时钟配置，需要在
	GPIO_Config函数之前完成，关于系统时钟如何配置，请参考对应用户手册或者
	RCC库函数及RCC相关例程。

适用芯片:
	HK32F103xC,HK32F103xD,HK32F103xE,HK32F103xxxxA全系列

涉及外设:
	I2S
	GPIO
	
涉及管脚:	
	MASTER
 	I2S1_PIN_WS            GPIOA_Pin_4
 	I2S1_PIN_CK            GPIOA_Pin_5
  	I2S1_PIN_SD            GPIOA_Pin_7
	SLAVER
  	I2S2_PIN_WS           GPIOB_Pin_12
  	I2S2_PIN_CK            GPIOB_Pin_13
  	I2S2_PIN_SD            GPIOB_Pin_15

注意事项:
	1. 推荐使用MDK5.00及其以上版本编译本工程（MDK-ARM）
	          或者IAR8.22及其以上版本编译本工程(EWARM);
	2. 使用之前检测电源VCC、GND等是否正常。


如何使用:
	1. 直接编译程序下载到评估版，重启评估版;
