******************************例程详细描述*****************************
功能描述:
	本例程演示了嵌套矢量中断控制器(NVIC)的使用，IRQ通道配置和如何屏蔽/激活不同的IRQ。
		1. 配置TIM2/TIM3/TIM4定时器，在每个定时器计数器更新事件时产生一个中断，周期分别
		为1，2，3s。
		2. TIM2/TIM3/TIM4抢占优先级分别为0，1，2；在中断中分别翻转LED1，LED2，LED3引脚。
		3. 当KEY2按下时，调用__disable_irq();关闭所有中断，此时所有LED IO不再翻转；当KEY2
		再次按下时调用__enable_irq();恢复中断，此时所有LED IO继续翻转
		4. 当KEY1按下时，调用__set_BASEPRI(0x40);屏蔽TIM3/4中断；此时只有TIM2中断继续
		触发，只LED1 IO翻转，当KEY1再次按下时，调用__set_BASEPRI(0x00)恢复所有中断，
		此时所有LED IO继续翻转
	
时钟配置:
	本例程采用HSE+PLL72MHz作为系统时钟，用户如果需要更改系统时钟配置，请参考对应用户手册或者
	RCC库函数及RCC相关例程。

适用芯片:
	HK32F103xC,HK32F103xD,HK32F103xE,HK32F103xxxxA全系列

涉及外设:
	EXTI/GPIO/NVIC/RCC/USART
	
涉及管脚:	
	PA0 ---------> KEY1
	PA1 ---------> KEY2
	PB1 ---------> LED1
	PB0 ---------> LED2
	PB5 ---------> LED3

注意事项:
	1. 推荐使用MDK5.00及其以上版本编译本工程（MDK-ARM）
	          或者IAR8.22及其以上版本编译本工程(EWARM);
	2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
	1. 直接编译程序下载到评估版，重启评估版;
	2. 上电后观察LED1/2/3闪烁状态是否正确，KEY1/2按下后，再观察LED1/2/3
	闪烁状态是否正确(该例程中按键和EXTI均未考虑软件滤波，用户使用时需注意这一点)
