******************************例程详细描述*****************************
功能描述:
    本例程描述了DMA的内存到内存到内存传输，传输的开始是由软件触发。源地址和目标地址递增。
通过设置DMA1通道6的通道使能位来开始传输。在传输结束时，会产生一个传输完全中断，在中断
函数中对传输完全标记(TC)进行清除。在主程序中会对源缓冲区和目标缓冲区进行比较，检查所有
数据是否已被正确传输。
    如果全部传输正确，则LED1灯会闪烁；
    如果不正确，则LED1会常亮。
        
时钟配置:
    本例程采用HSE72MHz作为系统时钟，用户如果需要更改系统时钟配置，可通过SYSCLK_SOURCE宏定义来选择不同的时钟源。

适用芯片:
	HK32F103xC,HK32F103xD,HK32F103xE,HK32F103xxxxA全系列

涉及外设:
	DMA/RCC/GPIO
	
涉及管脚:   
    LED1： PB1

注意事项:
	1. 推荐使用MDK5.00及其以上版本编译本工程（MDK-ARM）
	          或者IAR8.22及其以上版本编译本工程(EWARM);
	2. 使用之前检测电源VCC、GND等是否正常;

如何使用:
	1. 直接编译程序下载到评估版，重启评估版;
	2. 程序上电会自动进行测试，测试通过则LED1会闪烁，表明DMA从FLASH到SRAM搬运正确，
       如果LED1常量则表示DMA从FLASH到SRAM搬运异常。
