******************************例程详细描述*****************************
功能描述:
	本例程描述了如何将系统时钟配置为8MHz、24MHz、36MHz、48MHz、56MHz、72MHz。
	通过调用库函数RCC_SYSCLKConfig()来完成对系统时钟的配置，可以在main.h文件中选择系统时钟，
	本例程还处理高速外部时钟（HSE）故障检测：当HSE时钟消失时（外部晶振损坏或断开），
	HSE、PLL被禁用（但PLL配置没有变化），HSI被选为系统时钟源，并产生中断（NMI）。
	在NMI中断中，打开HSE以及HSERDY中断，一旦HSE恢复，则马上生成HSERDY中断，在中断中将系统时钟恢复到故障之前的状态。
	LED2不断翻转表明固件正常运行。
	
时钟配置:
	本例程采用HSI8MHz作为系统时钟，用户如果需要更改系统时钟配置，请参考对应用户手册或者
	RCC库函数及RCC相关例程。

适用芯片:
	HK32F103xC,HK32F103xD,HK32F103xE,HK32F103xxxxA全系列

涉及外设:
	RCC/GPIO/NVIC
	
涉及管脚:	
	PA8 ---------> MCO输出pin
	PB0 ---------> LED2

注意事项:
	1. 推荐使用MDK5.00及其以上版本编译本工程（MDK-ARM）
	          或者IAR8.22及其以上版本编译本工程(EWARM);
	2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
	1. 直接编译程序下载到评估版，重启评估版;
	2. 在main.h文件中选择系统时钟频率，观察PA8 pin上的时钟输出。
