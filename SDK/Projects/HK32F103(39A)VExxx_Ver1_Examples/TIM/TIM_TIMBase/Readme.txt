******************************例程详细描述*****************************
功能描述:
	本例程展示如何在输出比较定时模式下配置TIM外围设备以及每个通道的相应中断请求，以便生成4个不同的时基
	1. 配置TIM2的4个通道为冻结输出模式，关闭CCR寄存器的预装载功能，开启比较中断；
	2. 每次进中断,同一通道都会累加相同的数据给CCR，并立即生效,这样保证单个通道每次进中断的时间相同；
	3. 在中断中翻转相应的指示引脚,然后通过示波器查看相应指示引脚的PWM频率。
	

时钟配置:
	本例程采用HSE72MHz作为系统时钟，用户如果需要更改系统时钟配置, 请参考对应用户手册或者
	RCC库函数及RCC相关例程。

适用芯片:
	HK32F103xC,HK32F103xD,HK32F103xE,HK32F103xxxxA全系列

涉及外设:
	TIM,GPIO
	
涉及管脚:	
	指示引脚PC6,PC7,PC8,PC9

注意事项:
	1. 推荐使用MDK5.00及其以上版本编译本工程（MDK-ARM）
	    或者IAR8.22及其以上版本编译本工程(EWARM);
	2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
	1. 直接编译程序下载到评估版，重启评估版;
	2. 用示波器查看指示引脚PC6,PC7,PC8,PC9的PWM输出，频率依次为73.24Hz，109.8Hz，219.7Hz, 439.4Hz。
