******************************例程详细描述*****************************
功能描述:
	本例程展示了在定时器输入引脚中接收到外部信号的上升沿后，如何使用TIM外围设备生成单脉冲模式
	1. 配置TIM4为触发从模式，触发输入通道为TIM4_CH2；
                2. 配置TIM4_CH1为输出单脉冲；	   
	3. TIM4_CH2收到来自外部上升沿信号，便打开TIM4的计数器，从而使TIM4_CH1输出单脉冲。
	
时钟配置:
	本例程采用HSE72MHz作为系统时钟，用户如果需要更改系统时钟配置, 请参考对应用户手册或者
	RCC库函数及RCC相关例程。

适用芯片:
	HK32F103xC,HK32F103xD,HK32F103xE,HK32F103xxxxA全系列

涉及外设:
	TIM,GPIO
	
涉及管脚:	
	TIM4_CH1(PB6),    TIM4_CH2(PB7)

注意事项:
	1. 推荐使用MDK5.00及其以上版本编译本工程（MDK-ARM）
	          或者IAR8.22及其以上版本编译本工程(EWARM);
	2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
	1. 直接编译程序下载到评估版，重启评估版;
	2. 给PB7输入一个上升沿信号（可以将PB7连接KEY1按键的引脚上（PA0），松开按键便可以才产生一个上升沿）
	3. 用示波器查看TIM4_CH1(PB6)会产生一个单脉冲，脉宽为2.048ms。
