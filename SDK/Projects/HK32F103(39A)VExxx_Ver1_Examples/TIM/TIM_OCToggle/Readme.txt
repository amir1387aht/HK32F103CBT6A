******************************例程详细描述*****************************
功能描述:
	本例程展示如何配置TIM3外围设备以生成具有四个不同频率的四个不同信号
	1. 配置TIM3的4个通道为OCToggle输出模式，关闭CCR寄存器的预装载功能，开启比较中断；
	2. 每次进中断,同一通道都会累加相同的数据给CCR，并立即生效,这样保证单个通道每次进中断的时间相同；
	3. 这样就可以让每个通道生成占空比为50%的PWM,然后用示波器查看每个PWM的频率。
	

时钟配置:
	本例程采用HSE72MHz作为系统时钟，用户如果需要更改系统时钟配置, 请参考对应用户手册或者
	RCC库函数及RCC相关例程。

适用芯片:
	HK32F103xC,HK32F103xD,HK32F103xE,HK32F103xxxxA全系列

涉及外设:
	TIM,GPIO
	
涉及管脚:	
	TIM3_CH1(PA6),TIM3_CH2(PA7),TIM3_CH3(PB0),TIM3_CH4(PB1)

注意事项:
	1. 推荐使用MDK5.00及其以上版本编译本工程（MDK-ARM）
	    或者IAR8.22及其以上版本编译本工程(EWARM);
	2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
	1. 直接编译程序下载到评估版，重启评估版;
	2. 用示波器查看PA6,PA7,PB0,PB1输出的PWM频率依次为183.1, 366.2, 732.4, 1464.8Hz。
