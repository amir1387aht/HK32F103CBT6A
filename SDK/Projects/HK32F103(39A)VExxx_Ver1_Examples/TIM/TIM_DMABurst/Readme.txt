******************************例程详细描述*****************************
功能描述:
	本例程展示了如何使用TIM1 DMA突发特性更新TIM1通道1周期和占空比
	1. 配置TIM1的计数器更新会触发DMA请求；
                2. DMA会把内存中的3个数据（0XFFF,0X000,0X555）依次搬进TIM1_DMAR寄存器暂存，	   
	3. 由于设置TIM1_DCR寄存器的DBA位为  从TIM1_ARR寄存器开始。设置TIM1_DCR寄存器的DBL位的长度3
	4. 所以3个数据依次被搬进了TIM1_ARR,TIM1_RCR,TIM1_CCR1.从而改变了原有的周期和占空比。
	
时钟配置:
	本例程采用HSE72MHz作为系统时钟，用户如果需要更改系统时钟配置, 请参考对应用户手册或者
	RCC库函数及RCC相关例程。

适用芯片:
	HK32F103xC,HK32F103xD,HK32F103xE,HK32F103xxxxA全系列

涉及外设:
	TIM,GPIO
	
涉及管脚:	
	TIM1_CH1(PA8),   

注意事项:
	1. 推荐使用MDK5.00及其以上版本编译本工程（MDK-ARM）
	          或者IAR8.22及其以上版本编译本工程(EWARM);
	2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
	1. 直接编译程序下载到评估版，重启评估版;
	2. 用示波器查看TIM1的通道1（PA8）的周期为170us，占空比 为33.3%。
	3. 连接串口，串口打印如下
	   TIM1_ARR=0xFFF
                   TIM1_RCR=0x0
                   TIM1_CCR1=0x555。
