******************************例程详细描述*****************************
功能描述:
	本例程展示了如何使用TIM外围设备来测量外部信号的频率和占空比。
	1. 配置TIM2输出占空比为50%，频率为72kHz的PWM波；
	2. 配置TIM3为PWM输入捕获 模式；
	3. 用TIM3来捕获TIM2输出的PWM,计算捕获的占空比和频率，是否与输出的一致。

时钟配置:
	本例程采用HSE72MHz作为系统时钟，用户如果需要更改系统时钟配置, 请参考对应用户手册或者
	RCC库函数及RCC相关例程。

适用芯片:
	HK32F103xC,HK32F103xD,HK32F103xE,HK32F103xxxxA全系列

涉及外设:
	TIM,GPIO,USART
	
涉及管脚:	
	TIM2_CH2(PA1),    TIM3_CH2(PA7), USART1_TX(PA9), USART1_RX(PA10)

注意事项:
	1. 推荐使用MDK5.00及其以上版本编译本工程（MDK-ARM）
	    或者IAR8.22及其以上版本编译本工程(EWARM);
	2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
	1. 直接编译程序下载到评估版，重启评估版;
	2. 用杜邦线连接PA1和PA7。
	3.连接串口工具，串口打印下面占空比和频率
	Duty cycle = 49.849548
                Frequency = 72216.000000
